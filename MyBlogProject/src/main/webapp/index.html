Blog World App

Manage Your Posts





This App is the perfect blog editor. It is designed with everything you need to manage, write, update, publish and share posts.
Be more productive and efficient with this straightforward tool - because at the end of the day, we all have something to share.



MAIN FEATURES

* Create your own login account
* Preview your posts before publishing
* Check and reply to comments
* Sort your posts by title, created/updated dates or comments. (Coming soon)
* Create scheduled posts for later publishing. (Coming soon)


RICH TEXT EDITOR (These new features coming soon)

* Format text
* Add pictures from the web
* Control picture size live by click and drag


SUPPORT

We have a support team 24/7. Send us an email and we will address your questions and concerns in brief.


INFORMATION

This App was created as a java web application project using Blogger API.
